# txt2excel

### 运行环境

需要安装python环境,并且配有如下库:

- os
- xlwt
- argparse
- numpy

### 使用方式

#### 1. 收集信息

要求按照如下格式收集接龙信息:

```
1. [编号]+[姓名]+[所在地]+[健康状况]+[温度]
```

示例:

```
1. 2019200810+刘嘉润+江苏徐州+良好+36.5度
```

分隔符一定不要搞错了,搞错了会报错, 最前面的序号微信会自动给

#### 2. 复制信息至文本文件

#### 3. 文本转excel

在终端中运行如下命令将文本文件转成excel

```
python main.py -i [文本文件路径] -o [输出文件路径.xls]
```

示例:

```
python main.py -i ./text.txt -o ./result.xls
```
